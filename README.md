## **about the project**
this website is a personal portfolio for the dj and artist known as buny. 

# **purpose**
this website is designed to be a "one-stop shop" for everything related to buny, focusing on:

live shows: real-time announcements and direct links to ticket platforms
multimedia: showcasing latest music releases and other projects
social connectivity: links to all social media channels
professional EPK: resources for promoters, featuring bio, past show history, and discography

## **tech stack**
this web app has been created with HTML and CSS.
